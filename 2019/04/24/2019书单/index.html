2019书单

消磨时间类

  • [x]《嫌疑人X的献身》

    看看盛名之下，是不是难负。

  • [ ] 《美丽新世界》

    反乌托邦

  • [ ] 《剑桥简明金庸武侠史》

    新垣平的作品，他还有个臭名昭著的名字宝树。《三体》出不了续集就是他给毁的，但这并不代表这本书不幽默诙谐。

  • [ ] 《巨人的陨落》（第二部）（第三部）

    其实还有6本没看掉。

  • [ ] 《男人来自火星，女人来自金星》

    老书了，当作心灵鸡汤吧。

  • [ ] 《番茄工作法图解》

    看下如何才能和自己的workflow结合在一起

  • [ ] 《三体》（英文版）

    刘宇昆翻译的，读完也是遥遥无期了。

  • [ ] 《疯癫与文明》

    可推荐的。但我略翻阅了几页，呃呃，并不认为我有能力精读此书

  • [ ] 《红星闪耀中国》

    斯诺写的，我党得天下民心所向。

  • [ ] 《天才在左，疯子在右》

    老书了。大烂书，不看了。

  • [ ] 《菊与刀》

    老书了。

  • [ ] 《乌合之众》

    读心理学的同学说这本书全是扯淡，我倒要看看怎么扯淡

  • [ ] 《礼物的流动》

    民族志实在是太有意思了，读懂它可以对当代农村现状管窥一豹。

  • [ ] 《小王子》

    高中送我最好的朋友，但自己从来没读过。

  • [ ] 《亲密关系》

    这本书真的可以反复读。

  • [ ] 《陶庵梦忆》

    张岱的游记，实在是有趣，成书于国破家亡之后。我最喜欢的《湖心亭看雪》就出于此。

  • [ ] 《变形记》

    表现主义的书，我大概是读不懂了。

  • [X] 《流浪地球制作手记》

  • [X] 《乡村教师》

  • [X] 《流浪地球》
